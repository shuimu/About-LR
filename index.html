关于LR（线性回归），有蛮多自己的思考的，现在一一记录下来，作为感悟。

线性回归简单看来，就是最小化损失函数。损失函数可以有很多的形式，比如：Loss = sum[ (f(xi) - y)^2 for i in (1, n) ] 这样的形式是符合最大似然的形式。 假设f(xi) = wx+b的形式，即线性的形式，那么 Loss = sum[ (wx+b-y)^2 for i in (1, n) ]，如果采用sgd的优化方法，对其求偏导得到： 对w求偏导：Loss'w = (wx+b-y)x 对b求偏导：Loss'b = (wx+b-y) 那么得到更新公式： w' = w + Loss'walpha b' = b + Loss'balpha 其中，alpha是学习率，公式化简即： w' = w + (wx+b-y)xalpha b' = b + (wx+b-y)alpha 考虑多个feature的情况，就是： w1' = w1 + (wx+b-y)x1alpha w2' = w2 + (wx+b-y)x2alpha ... wn' = wn + (wn+b-y)xnalpha 其中，b'可以看做xi=1的情况。 考虑成为向量的情况， w = w + (wx+b-y)x*alpha
